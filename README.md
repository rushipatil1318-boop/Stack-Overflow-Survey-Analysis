# Stack Overflow Developer Survey Analysis

Analyzed the Stack Overflow annual developer survey to understand 
what the global developer community looks like — which languages 
are popular, how salaries vary, and what remote work trends look like.

## Why this project

Stack Overflow releases their survey data publicly every year and 
it's one of the most honest looks at what developers actually use 
and earn. I wanted to see how the numbers break down across 
countries, experience levels, and education backgrounds.

## What I looked at

- Most popular programming languages globally
- Salary differences across countries and experience levels
- Remote work preferences post-COVID
- How education background affects developer salaries
- Job satisfaction trends

## Key findings

- Python and JavaScript are consistently the most used languages
- US developers earn significantly more than European counterparts
- Developers with 5+ years experience earn roughly 2x entry level
- Remote work preference stayed high even after COVID restrictions lifted
- Formal degree holders don't always out-earn self-taught developers

## Tools used

Python · Pandas · Matplotlib · Seaborn · NumPy · Jupyter Notebook

## Data source

Stack Overflow Annual Developer Survey — publicly available at 
stackoverflow.com/research/developer-survey
